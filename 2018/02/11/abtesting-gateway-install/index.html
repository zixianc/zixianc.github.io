配置文件说明

关键配置文件为nginx.conf、default.conf、upstream.conf、vhost.conf

  • nginx.conf为nginx入口配置文件,在这里引入包含其他配置文件
  • default.conf为管理后台配置文件
  • upstream.conf为upstream配置文件
  • vhost.conf为代理转发配置文件模板
